# Newton Screen Recorder for Android

تطبيق أندرويد لتسجيل:
- شاشة الهاتف.
- صوت الهاتف الداخلي (Android 10+، حسب سماح التطبيق المصدر).
- صوت الميكروفون.
- تقليل الضوضاء وإلغاء الصدى والتحكم التلقائي في مستوى الصوت عند دعم الجهاز.

## المتطلبات
- Android 10 أو أحدث.
- منح إذن تسجيل الشاشة والميكروفون.
- بعض التطبيقات المحمية تمنع تسجيل صوتها أو صورتها.

## بناء APK من GitHub
1. ارفع محتويات هذا المشروع إلى مستودع GitHub جديد.
2. افتح تبويب **Actions**.
3. شغّل **Build Android APK** أو انتظر تشغيله تلقائيًا.
4. بعد نجاح البناء افتح التشغيل، ثم حمّل Artifact باسم:
   `Newton-Screen-Recorder-APK`
5. فك الضغط وثبّت `app-debug.apk`.
